Bases: GObject.Object, Gio.AsyncInitable, Gio.Initable, Polkit.Subject

An object that represents an user session.

The session id is an opaque string obtained from ConsoleKit.

static new_for_process(pid, cancellable, callback, *user_data)
Parameters:
  • pid (int) – The process id of the process to get the session for.
  • cancellable (Gio.Cancellable or None) – A Gio.Cancellable or None.
  • callback (Gio.AsyncReadyCallback or None) – A Gio.AsyncReadyCallback to call when the request is satisfied
  • user_data (object) – The data to pass to callback.

Asynchronously creates a new Polkit.UnixSession object for the process with process id pid.

When the operation is finished, callback will be invoked in the thread-default main loop of the thread you are calling this method from. You can then call Polkit.UnixSession.new_for_process_finish () to get the result of the operation.

This method constructs the object asynchronously, for the synchronous and blocking version use Polkit.UnixSession.new_for_process_sync ().
