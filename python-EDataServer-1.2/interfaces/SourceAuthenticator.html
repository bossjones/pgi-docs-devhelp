get_prompt_strings(source)
Parameters:source (EDataServer.Source) – an EDataServer.Source
Returns:
prompt_title:the title of the prompt
prompt_message:the prompt message for the user
prompt_description:
 the detailed description of the prompt
Return type:(prompt_title: str, prompt_message: str, prompt_description: str)

Generates authentication prompt strings for source.

For registry service clients, EDataServer.SourceRegistry calls this function as part of EDataServer.SourceRegistry.authenticate_sync (). In the registry service itself, #EAuthenticationSession calls this function during initialization. This function should rarely need to be called explicitly outside of those two cases.

The #ESourceAuthenticatorInterface defines a default behavior for this method which should suffice in most cases. But implementors can still override the method if needed for special circumstances.

Free each of the returned prompt strings with GLib.free ().

New in version 3.6.

get_without_password()
Returns:whether to try to authenticate without asking for the password
Return type:bool

Returns whether the used authentication method can be used without a password prompt. If so, then user is not asked for the password, only if the authentication fails. The default implementation returns False, which means always asks for the password (or read it from a keyring).

New in version 3.10.

try_password(password, cancellable, callback, *user_data)
Parameters:
  • password (GLib.String) – a user-provided password
  • cancellable (Gio.Cancellable or None) – optional Gio.Cancellable object, or None
  • callback (Gio.AsyncReadyCallback or None) – a Gio.AsyncReadyCallback to call when the request is satisfied
  • user_data (object) – data to pass to the callback function

Asyncrhonously attempts to authenticate using password.

The password is passed in a GLib.String container so its content is not accidentally revealed in a stack trace.

When the operation is finished, callback will be called. You can then call EDataServer.SourceAuthenticator.try_password_finish () to get the result of the operation.

New in version 3.6.

try_password_finish(result)
Parameters:result (Gio.AsyncResult) – a Gio.AsyncResult
Raises:GLib.GError
Returns:the authentication result
Return type:EDataServer.SourceAuthenticationResult

Finishes the operation started with EDataServer.SourceAuthenticator.try_password ().

If an error occurred, the function sets error and returns EDataServer.SourceAuthenticationResult.ERROR.
