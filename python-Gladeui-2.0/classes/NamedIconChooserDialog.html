Signal Details

Gladeui.NamedIconChooserDialog.signals.icon_activated(named_icon_chooser_dialog)
Signal Name:icon-activated
Flags:RUN_LAST
Parameters:named_icon_chooser_dialog (Gladeui.NamedIconChooserDialog) – The object which received the signal

This signal is emitted when the user “activates” an icon in the named icon chooser. This can happen by double-clicking on an item in the recently used resources list, or by pressing Enter.

Gladeui.NamedIconChooserDialog.signals.selection_changed(named_icon_chooser_dialog)
Signal Name:selection-changed
Flags:RUN_LAST
Parameters:named_icon_chooser_dialog (Gladeui.NamedIconChooserDialog) – The object which received the signal

This signal is emitted when there is a change in the set of selected icon names. This can happen when a user modifies the selection with the mouse or the keyboard, or when explicitely calling functions to change the selection.
