class Json.Array

A JSON array type. The contents of the Json.Array structure are private and should only be accessed by the provided API

foreach_element(func, *data)
Parameters:
  • func (Json.ArrayForeach) – the function to be called on each element
  • data (object) – data to be passed to the function

Iterates over all elements of array and calls func on each one of them.

It is safe to change the value of a Json.Node of the array from within the iterator func, but it is not safe to add or remove elements from the array.

New in version 0.8.

Removes the Json.Node inside array at index_ freeing its allocated resources.

unref()

Decreases by one the reference count of a Json.Array. If the reference count reaches zero, the array is destroyed and all its allocated resources are freed.
