peek_real_data(ret_size)
Parameters:ret_size (int) – Location to return the size of the returned data buffer.
Raises:GLib.GError
Returns:Pointer to the real clipboard data. The size in bytes of this buffer is returned in the ret_size argument.
Return type:object

Queries a pointer directly to the clipboard data of a Gsf.ClipData. The resulting pointer is not necessarily the same data pointer that was passed to Gsf.Blob.new () prior to creating the clip_data. For example, if the data is in Gsf.ClipFormat.WINDOWS_CLIPBOARD format, then it will have extra header bytes in front of the actual metafile data. This function will skip over those header bytes if necessary and return a pointer to the “real” data.
