Return type:bytes

Read num_bytes. Does not change the current position if there is an error. Will only read if the entire amount can be read.

Note

This function is an alternative implementation for bindings. The following text is the documentation of the original, replaced function, which might include additional information:

Read at least num_bytes. Does not change the current position if there is an error. Will only read if the entire amount can be read. Invalidates the buffer associated with previous calls to Gsf.Input.read.
