get_file_info()
Returns:a Gio.FileInfo, or None
Return type:Gio.FileInfo

Retrieves the Gio.FileInfo constructed from the GZIP header data of compressed data processed by compressor, or None if decompressor ‘s Gio.ZlibDecompressor :format property is not Gio.ZlibCompressorFormat.GZIP, or the header data was not fully processed yet, or it not present in the data stream at all.
