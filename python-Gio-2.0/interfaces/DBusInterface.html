Class Details

class Gio.DBusInterface

Bases: GObject.GInterface

The Gio.DBusInterface type is the base type for D-Bus interfaces both on the service side (see Gio.DBusInterfaceSkeleton ) and client side (see Gio.DBusProxy ).

New in version 2.30.

get_info()
Returns:A Gio.DBusInterfaceInfo. Do not free.
Return type:Gio.DBusInterfaceInfo

Gets D-Bus introspection information for the D-Bus interface implemented by interface_.

New in version 2.30.

get_object()
Returns:A Gio.DBusObject or None. The returned reference should be freed with GObject.Object.unref ().
Return type:Gio.DBusObject

Gets the Gio.DBusObject that interface_ belongs to, if any.

Note

This function is an alternative implementation for bindings. The following text is the documentation of the original, replaced function, which might include additional information:

Gets the Gio.DBusObject that interface_ belongs to, if any.

It is not safe to use the returned object if interface_ or the returned object is being used from other threads. See Gio.DBusInterface.get_object () for a thread-safe alternative.

New in version 2.32.

set_object(object)
Parameters:object (Gio.DBusObject or None) – A Gio.DBusObject or None.

Note that interface_ will hold a weak reference to object.
