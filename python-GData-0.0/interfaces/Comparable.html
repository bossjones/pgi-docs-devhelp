compare(other)
Parameters:other (GData.Comparable or None) – another GData.Comparable of the same type, or None
Returns:True on success, False otherwise
Return type:int

Compares the two objects, returning -1 if self is “less than” other by some metric, 0 if they’re equal, or 1 if self is “greater than” other.

None values are handled gracefully, with 0 returned if both self and other are None, -1 if self is None and 1 if other is None.

The other object must be of the same type as self, or of a type derived from self ‘s type.

New in version 0.7.0.
