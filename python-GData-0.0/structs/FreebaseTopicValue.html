class GData.FreebaseTopicValue

Opaque struct containing a value of a Freebase topic object. This struct may contain a simple value (integers, doubles, strings...) or complex values, expressed through a GData.FreebaseTopicObject.

New in version 0.15.1.

copy_value()
Returns:an empty GObject.Value
Return type:gvalue: GObject.Value

Copies in gvalue the value held in value. the GObject.Value must be later freed through GObject.Value.unset ()

New in version 0.15.1.

get_creator()
Returns:the creator of this value, as a Freebase ID
Return type:str

Returns the Freebase ID of the user that created this value.

New in version 0.15.1.

get_text()
Returns:a textual representation of value

get_timestamp()
Returns:The creation time of value, or -1
Return type:int

Returns the time at which this value was created in the Freebase database. It’s a UNIX timestamp in seconds since the epoch. If value has no timestamp, -1 will be returned.

New in version 0.15.1.

get_value_type()
Returns:the GObject.Type of the contained value
Return type:GObject.GType

Returns the GObject.Type of the real value held in value.

New in version 0.15.1.

is_image()
Returns:Whether value holds a Freebase image object
Return type:bool

Returns true if value holds a freebase image object, on such values it will be valid to call GData.FreebaseService.get_image () to get a stream to the image itself.

New in version 0.15.1.

ref()

unref()
