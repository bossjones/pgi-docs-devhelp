class Ggit.DiffFile

Represents a file in a Ggit.Diff.

Gets the #GgitDifflags for file.

Return type:int

Returns:the file’s Ggit.OId.
Return type:Ggit.OId

Gets the Ggit.OId for file.

get_path()
Returns:the file’s path, or None.
Return type:str

Gets the path of file or None if it is unknown.

get_size()
Returns:the file’s size.
Return type:int

Gets the size for file.

ref()
Returns:a Ggit.DiffFile.
Return type:Ggit.DiffFile

Atomically increments the reference count of file by one. This function is MT-safe and may be called from any thread.

unref()

Atomically decrements the reference count of file by one. If the reference count drops to 0, file is freed.
