class Ggit.MergeOptions

Represents the options used when merging.

static new(flags, rename_threshold, target_limit, metric, file_favor)
Parameters:
  • flags (Ggit.MergeTreeFlags) – flags to consider when merging.
  • rename_threshold (int) – similarity to consider a file renamed (default 50).
  • target_limit (int) – maximum similarity sources to examine (overrides the `merge.renameLimit` config) (default 200).
  • metric (Ggit.DiffSimilarityMetric or None) – a Ggit.DiffSimilarityMetric or None to use internal metric.
  • file_favor (Ggit.MergeFileFavor) – a Ggit.MergeFileFavor.

a newly allocated Ggit.MergeOptions.
