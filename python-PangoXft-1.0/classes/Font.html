class PangoXft.Font(**kwargs)

Bases: Pango.Font

PangoXft.Font is an implementation of #PangoFcFont using the Xft library for rendering. It is used in conjunction with PangoXft.FontMap.

static get_glyph(font, wc)
Parameters:
  • font (Pango.Font) – a Pango.Font for the Xft backend
  • wc (unicode) – Unicode codepoint to look up
Returns:

the glyph index, or 0, if the Unicode character does not exist in the font.
Return type:

int

Gets the glyph index for a given Unicode character for font. If you only want to determine whether the font has the glyph, use PangoXft.Font.has_char ().

Use pango_fc_font_get_glyph() instead.

New in version 1.2.
