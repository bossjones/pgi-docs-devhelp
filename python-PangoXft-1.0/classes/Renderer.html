Class Details

class PangoXft.Renderer(**kwargs)

Bases: Pango.Renderer

PangoXft.Renderer is a subclass of Pango.Renderer used for rendering with Pango’s Xft backend. It can be used directly, or it can be further subclassed to modify exactly how drawing of individual elements occurs.

New in version 1.8.

Pango.Renderer

Create a new PangoXft.Renderer to allow rendering Pango objects with the Xft library. You must call PangoXft.Renderer.set_draw () before using the renderer.
