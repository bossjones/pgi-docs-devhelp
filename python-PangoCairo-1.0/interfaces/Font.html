Class Details

class PangoCairo.Font

Bases: GObject.GInterface

PangoCairo.Font is an interface exported by fonts for use with Cairo. The actual type of the font will depend on the particular font technology Cairo was compiled to use.

New in version 1.18.

get_scaled_font()
Returns:the cairo.ScaledFont used by font, or None if font is None.
Return type:cairo.ScaledFont

Gets the cairo.ScaledFont used by font. The scaled font can be referenced and kept using cairo_scaled_font_reference().

New in version 1.18.
