Parameters:

  • callback (Gio.AsyncReadyCallback or None) – A Gio.AsyncReadyCallback to call when the request is satisfied or None.

Asynchronously invokes the ScanDevices() D-Bus method on proxy. When the operation is finished, callback will be invoked in the thread-default main loop of the thread you are calling this method from. You can then call ModemManager.GdbusOrgFreedesktopModemManager1.call_scan_devices_finish () to get the result of the operation.
