connect(properties, cancellable, callback, *user_data)
Parameters:
  • properties (ModemManager.SimpleConnectProperties) – A ModemManager.SimpleConnectProperties bundle.
  • cancellable (Gio.Cancellable or None) – A Gio.Cancellable or None.
  • callback (Gio.AsyncReadyCallback or None) – A Gio.AsyncReadyCallback to call when the request is satisfied or None.
  • user_data (object) – User data to pass to callback.

Asynchronously requests to connect the modem using the given properties.

When the operation is finished, callback will be invoked in the thread-default main loop of the thread you are calling this method from. You can then call ModemManager.ModemSimple.connect_finish () to get the result of the operation.

See ModemManager.ModemSimple.connect_sync () for the synchronous, blocking version of this method.
