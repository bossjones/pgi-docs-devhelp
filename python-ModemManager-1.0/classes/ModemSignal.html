setup(rate, cancellable, callback, *user_data)
Parameters:
  • rate (int) – Rate to use when refreshing signal values.
  • cancellable (Gio.Cancellable or None) – A Gio.Cancellable or None.
  • callback (Gio.AsyncReadyCallback or None) – A Gio.AsyncReadyCallback to call when the request is satisfied or None.
  • user_data (object) – User data to pass to callback.

Asynchronously setups the extended signal quality retrieval.

When the operation is finished, callback will be invoked in the thread-default main loop of the thread you are calling this method from. You can then call ModemManager.ModemSignal.setup_finish () to get the result of the operation.

See ModemManager.ModemSignal.setup_sync () for the synchronous, blocking version of this method.
