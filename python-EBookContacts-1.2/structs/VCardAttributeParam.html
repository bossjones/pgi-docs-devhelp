get_name()
Returns:The name of the parameter.
Return type:str

Gets the name of param.

For example, for the only parameter of the vCard attribute: TEL;TYPE=WORK,VOICE:(111) 555-1212 this would return TYPE (which is string-equivalent to %EVC_TYPE).

get_values()
Returns:A list of string elements representing the parameter’s values.
Return type:[str]

Gets the list of values from param. The list and its contents are owned by param, and must not be freed.

For example, for the TYPE parameter of the vCard attribute: TEL;TYPE=WORK,VOICE:(111) 555-1212 this would return the list WORK, VOICE.

remove_values()

Removes and frees all values from param.
