set_automatic_login(enabled)
Parameters:enabled (bool) – whether or not to autologin for user.

If enabled is set to True then this user will automatically be logged in at boot up time. Only one user can be configured to auto login at any given time, so subsequent calls to AccountsService.User.set_automatic_login () override previous calls.

Note this function is synchronous and ignores errors.

set_password_mode(password_mode)
Parameters:password_mode (AccountsService.UserPasswordMode) – a AccountsService.UserPasswordMode

Changes the password of user. If password_mode is AccountsService.UserPasswordMode.SET_AT_LOGIN then the user will be asked for a new password at the next login. If password_mode is AccountsService.UserPasswordMode.NONE then the user will not require a password to log in.

Note this function is synchronous and ignores errors.
