Gtk.RadioActionEntry

Fields

Name Type Access Description
accelerator str r/w The accelerator for the action, in the format understood by Gtk.accelerator_parse ().
label str r/w The label for the action. This field should typically be marked for translation, see Gtk.ActionGroup.set_translation_domain ().
name str r/w The name of the action.
stock_id str r/w The stock id for the action, or the name of an icon from the icon theme.
tooltip str r/w The tooltip for the action. This field should typically be marked for translation, see Gtk.ActionGroup.set_translation_domain ().
value int r/w The value to set on the radio action. See Gtk.RadioAction.get_current_value ().

Methods

None

Details

class Gtk.RadioActionEntry

Gtk.RadioActionEntry structs are used with Gtk.ActionGroup.add_radio_actions () to construct groups of radio actions.

Deprecated since version 3.10.
