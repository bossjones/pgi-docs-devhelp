Class Details

class Gtk.RecentAction(name, label, tooltip, stock_id, **kwds)

Bases: Gtk.Action, Gtk.RecentChooser

A Gtk.RecentAction represents a list of recently used files, which can be shown by widgets such as Gtk.RecentChooserDialog or Gtk.RecentChooserMenu.

To construct a submenu showing recently used files, use a Gtk.RecentAction as the action for a . To construct a menu toolbutton showing the recently used files in the popup menu, use a Gtk.RecentAction as the action for a element.

set_show_numbers(show_numbers)
Parameters:show_numbers (bool) – True if the shown items should be numbered

Sets whether a number should be added to the items shown by the widgets representing action. The numbers are shown to provide a unique character for a mnemonic to be used inside the menu item’s label. Only the first ten items get a number to avoid clashes.

New in version 2.12.

Deprecated since version 3.10.
