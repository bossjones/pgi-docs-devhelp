GstAllocators.dmabuf_allocator_alloc(allocator, fd, size)
Parameters:
  • allocator (Gst.Allocator or None) – allocator to be used for this memory
  • fd (int) – dmabuf file descriptor
  • size (int) – memory size
Returns:

a Gst.Memory based on allocator. When the buffer will be released dmabuf allocator will close the fd. The memory is only mmapped on gst_buffer_mmap() request.
Return type:

Gst.Memory

Return a Gst.Memory that wraps a dmabuf file descriptor.

New in version 1.2.
