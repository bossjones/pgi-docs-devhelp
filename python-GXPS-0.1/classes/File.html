get_document_for_link_target(target)
Parameters:target (GXPS.LinkTarget) – a GXPS.LinkTarget
Returns:the index of the document pointed by the given GXPS.LinkTarget or -1.
Return type:int

Gets the index of the document in xps pointed by target. If the GXPS.LinkTarget does not reference a document, or referenced document is not found in xps file -1 will be returned. In this case you can look for the page pointed by the link target by calling GXPS.Document.get_page_for_anchor () with the anchor of the GXPS.LinkTarget for every document in xps.
