get_page_size(n_page)
Parameters:n_page (int) – the index of a page in doc
Returns:True if the page size information is available in doc, False otherwise.
width:return location for the width of n_page
height:return location for the height of n_page
Return type:(bool, width: float, height: float)

Gets the typical size of the page at index n_page in doc document. This function is useful to get the advisory size of pages in a document without creating GXPS.Page objects. This page size might be different than the actual page size so page dimension might need to be updated once the page is loaded. Advisory page sizes are not always available in doc, in which case this function returns False. To get the authoritative size of a page you should use GXPS.Page.get_size () instead.
