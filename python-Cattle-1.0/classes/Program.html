Class Details

class Cattle.Program(**kwargs)

Bases: GObject.Object

Opaque data structure representing a program. It should never be accessed directly; use the methods below instead.

static new()
Returns:a new Cattle.Program
Return type:Cattle.Program

Create a new Cattle.Program.

A single instance of a program can be shared between multiple interpreters, as long as the object is not modified after it has been initialized.

get_input()
Returns:input for program
Return type:Cattle.Buffer

Get the input for program. See Cattle.Program.set_input ().

get_instructions()
Returns:the first instruction in program
Return type:Cattle.Instruction

Get the instructions for program. See Cattle.Program.load () and Cattle.Program.set_instructions ().

load(buffer)
Parameters:buffer (Cattle.Buffer) – a Cattle.Buffer containing the code
Raises:GLib.GError
Returns:True on success, False otherwise
Return type:bool

Load program from buffer.

The buffer can optionally contain also the input for the program: in that case, the input must be separated from the code by a bang (!) character.

In case of failure, error is filled with detailed information. The error domain is %CATTLE_ERROR, and the error code is from the Cattle.Error enumeration.

set_input(input)
Parameters:input (Cattle.Buffer) – input for program

Set the input for program.

If the size of input is zero, the program’s input will be retrieved at runtime.

set_instructions(instructions)
Parameters:instructions (Cattle.Instruction) – instructions for program

Set the instructions for program.

You shouldn’t usually need to use this: see Cattle.Program.load () for the standard way to load a program.
