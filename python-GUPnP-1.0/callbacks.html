Callbacks

ServiceIntrospectionCallback(info, introspection, error, user_data)
ServiceProxyActionCallback(proxy, action, user_data)
ServiceProxyNotifyCallback(proxy, variable, value, user_data)

Details

GUPnP.ServiceIntrospectionCallback(info, introspection, error, user_data)
Parameters:
  • info (GUPnP.ServiceInfo) – The GUPnP.ServiceInfo introspection was requested for
  • introspection (GUPnP.ServiceIntrospection) – The new GUPnP.ServiceIntrospection object, or None
  • error (GLib.GError) – The GLib.Error that occurred, or None
  • user_data (object) – User data

Callback notifying that introspection for info has been obtained.

GUPnP.ServiceProxyActionCallback(proxy, action, user_data)
Parameters:
  • proxy (GUPnP.ServiceProxy) – The GUPnP.ServiceProxy action is called from
  • action (GUPnP.ServiceProxyAction) – The GUPnP.ServiceProxyAction in progress
  • user_data (object) – User data

Callback notifying that action on proxy has returned and GUPnP.ServiceProxy.end_action () etc can be called.

GUPnP.ServiceProxyNotifyCallback(proxy, variable, value, user_data)
Parameters:
  • proxy (GUPnP.ServiceProxy) – The GUPnP.ServiceProxy the notification originates from
  • variable (str) – The name of the variable being notified
  • value (GObject.Value) – The GObject.Value of the variable being notified
  • user_data (object) – User data

Callback notifying that the state variable variable on proxy has changed to value.
