register_resource_proxy_type(upnp_type, type)
Parameters:
  • upnp_type (str) – The UPnP type name of the resource.
  • type (GObject.GType) – The requested GObject.Type assignment for the resource proxy.

Registers the GObject.Type type for the proxy of resource of UPnP type upnp_type. After this call, the factory factory will create object of GObject.Type type each time it is asked to create a resource proxy object for UPnP type upnp_type.

Note: GObject.Type type must be a derived type of #GUPNP_TYPE_DEVICE_PROXY if resource is a device or #GUPNP_TYPE_SERVICE_PROXY if its a service.

register_resource_type(upnp_type, type)
Parameters:
  • upnp_type (str) – The UPnP type name of the resource.
  • type (GObject.GType) – The requested GObject.Type assignment for the resource.

Registers the GObject.Type type for the resource of UPnP type upnp_type. After this call, the factory factory will create object of GObject.Type type each time it is asked to create a resource object for UPnP type upnp_type.

Note: GObject.Type type must be a derived type of #GUPNP_TYPE_DEVICE if resource is a device or #GUPNP_TYPE_SERVICE if its a service.
