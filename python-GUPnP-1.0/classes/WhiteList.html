check_context(context)
Parameters:context (GUPnP.Context) – A GUPnP.Context to test.
Returns:True if context is matching the white_list criterias, False otherwise.
Return type:bool

It will check if the context is allowed or not. The white_list will check all its entries againt GUPnP.Context interface, host ip and network fields information. This function doesn’t take into account the white_list status (enabled or not).

New in version 0.20.5.

clear()

Remove all entries from GLib.List that compose the white list. The list is now empty. Even if GUPnP.WhiteList is enabled, it will have the same behavior as if it was disabled.

New in version 0.20.5.
