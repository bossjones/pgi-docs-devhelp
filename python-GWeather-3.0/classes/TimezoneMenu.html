A Gtk.ComboBox subclass for choosing a GWeather.Timezone

static new(top)
Parameters:top (GWeather.Location) – the top-level location for the menu.
Returns:the new GWeather.TimezoneMenu
Return type:Gtk.Widget

Creates a new GWeather.TimezoneMenu.

top will normally be the location returned from GWeather.Location.get_world (), but you can create a menu that contains the timezones from a smaller set of locations if you want.

Gets menu ‘s timezone id.

set_tzid(tzid)
Parameters:tzid (str or None) – a tzdata id (eg, “America/New_York”)

Sets menu to the given tzid. If tzid is None, sets menu to “Unknown”.
