render_to_cairo(cairo, w, h)
Parameters:
  • cairo (cairo.Context) – a cairo context
  • w (float) – width
  • h (float) – height

Renders a graph using cairo. w and h are the requested width an height of the rendered graph, in the current cairo coordinate space. This function is not suited for multiple rendering of the same graph. gog_renderer_render_to cairo or GOffice.gRenderer.update /gog_renderer_get_cairo_surface should be used instead.
