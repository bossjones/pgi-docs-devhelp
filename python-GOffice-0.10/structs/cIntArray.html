class GOffice.cIntArray

FocIntArray ::n is the size of the array. A boxed type used to hold an array of integers.

New in version 0.8.2.

static new(n)
Parameters:n (int) – the number of integers in the array.
Returns:the newly created GOffice.cIntArray with an initial references GOffice.count of 1.
Return type:GOffice.cIntArray

Creates a new GOffice.cIntArray instances with n values initialized to 0. The values can be changed using direct access:

GocIntArray array = goc_int_array_new (2);
array->vals[0] = my_first_int;
array->vals[1] = my_second_int;

New in version 0.8.2.

ref()
Returns:the referenced GOffice.cIntArray.
Return type:GOffice.cIntArray

Increases the references GOffice.count of array by 1.

New in version 0.8.2.

unref()

Decreases the references GOffice.count of array by 1, and destroys it if the references GOffice.count becomes 0.

New in version 0.8.2.
