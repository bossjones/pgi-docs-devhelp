find_batch_by_source(source)
Parameters:source (RB.Source) – the RB.Source to search for
Returns:GLib.List of RB.TrackTransferBatch objects, not referenced
Return type:[RB.TrackTransferBatch]

Finds all transfer batches where source is the source or destination. This should be used to wait for transfers to finish (or cancel them) before ejecting a device. The transfer batches are returned in the order they’re found in the queue, so waiting for the RB.TrackTransferBatch ::complete signal on the last one is sufficient to wait for them all to finish.

start_batch(batch)
Parameters:batch (RB.TrackTransferBatch) – the RB.TrackTransferBatch to add to the queue

Adds a new transfer batch to the transfer queue; if the queue is currently empty, the transfer will start immediately, but not before the call returns.
