service-type str r/w D-Bus service name of the VPN plugin that this setting uses to connect to its network. i.e. org.freedesktop.NetworkManager.vpnc for the vpnc plugin.
user-name str r/w If the VPN connection requires a user name for authentication, that name should be provided here. Type:str
Default Value:None

If the VPN connection requires a user name for authentication, that name should be provided here. If the connection is available to more than one user, and the VPN requires each user to supply a different name, then leave this property empty.
