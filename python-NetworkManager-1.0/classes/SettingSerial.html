Name Type Flags Short Description
baud int r/w/c Speed to use for communication over the serial port. Note that this value usually has no effect for mobile broadband modems as they generally ignore speed settings and use the highest available speed.
bits int r/w/c Byte-width of the serial communication. The 8 in ‘8n1’ for example.
parity   r/w/c Parity setting of the serial port. Either ‘E’ for even parity, ‘o’ for odd parity, or ‘n’ for no parity.
send-delay int r/w/c Time to delay between each byte sent to the modem, in microseconds.
stopbits int r/w/c Number of stop bits for communication on the serial port. Either 1 or 2. The 1 in ‘8n1’ for example.
