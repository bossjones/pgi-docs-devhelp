Parameters:reason (str) – reason to pause
Raises:GLib.GError
Returns:The pause cookie ID.
Return type:int

Asks miner to pause. On success the cookie ID is returned, this is what must be used in TrackerMiner.Miner.resume () to resume operations. On failure error will be set and Tracker.DBUS_OBJECT_STEROIDS-1 will be returned.

Asks the miner to resume processing. The cookie must be something returned by TrackerMiner.Miner.pause (). The miner won’t actually resume operations until all pause requests have been resumed.

TrackerMiner.Miner.signals.progress(miner, status, progress, remaining_time)
Signal Name:

progress
Flags:

RUN_LAST
Parameters:
  • miner (TrackerMiner.Miner) – The object which received the signal
  • status (str) – miner status
  • progress (float) – a float indicating miner progress, from 0 to 1.
  • remaining_time (int) – a int indicating the reamaining processing time, in seconds.

the Tracker.DBUS_OBJECT_STEROIDS ::progress signal will be emitted by TrackerMiner.Miner implementations to indicate progress about the data mining process. status will contain a translated string with the current miner status and progress will indicate how much has been processed so far. remaining_time will give the number expected of seconds to finish processing, 0 if the value cannot be estimated, and Tracker.DBUS_OBJECT_STEROIDS-1 if its not applicable.

New in version 0.12.
