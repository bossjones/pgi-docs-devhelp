Return type:str

The unique string identifying this TrackerMiner.Decorator that has extracted the extended metadata. This is essentially an identifier so it’s clear WHO has extracted this extended metadata.

next(cancellable, callback, *user_data)
Parameters:
  • cancellable (Gio.Cancellable or None) – a Gio.Cancellable.
  • callback (Gio.AsyncReadyCallback or None) – a Gio.AsyncReadyCallback.
  • user_data (object) – user_data for callback.

Processes the next resource in the queue to have extended metadata extracted. If the item in the queue has been completed already, it signals it’s completion instead.

This function will give a GLib.Error if the miner is paused at the time it is called.

New in version 0.18.
