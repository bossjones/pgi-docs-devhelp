Adds a pane, with optional Anjuta.CommandBar entries, to an Anjuta.Dock. This method adds a pane with no grip that cannot be closed, floating or iconified.

  • pane_label (str) – Label to display in this pane’s grip
  • stock_icon (str) – Stock icon to display in this pane’s grip
  • pane (Anjuta.DockPane) – The Anjuta.DockPane to add to the dock. The dock takes ownership of the pane object.
  • placement (Gdl.DockPlacement) – A Gdl.DockPlacement value indicating where the pane should be placed
  • entries ([Anjuta.CommandBarEntry] or None) – Anjuta.CommandBar entries for this pane. Can be None
  • user_data (object) – User data to pass to the entry callback

Adds a pane, with optional Anjuta.CommandBar entries, to an Anjuta.Dock. This method adds a pane with no grip that cannot be closed, floating or iconified. If there was an old command pane, that pane is removed in favour of the new pane.

set_command_bar(command_bar)
Parameters:command_bar (Anjuta.CommandBar) – An Anjuta.CommandBar to associate with this dock

Associates an Anjuta.CommandBar with this dock. Command bars can be used to provide different sets of commands based on the currently visible pane.
