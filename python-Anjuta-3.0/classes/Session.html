get_file_from_relative_uri(uri, fragment)
Parameters:
  • uri (str) – a relative URI from a key
  • fragment (str or None) – fragment part of the URI if existing, can be None
Returns:

A new Gio.File that has to be freed with GObject.Object.unref ().
Return type:

Gio.File

Return a Gio.File corresponding to the URI and and optional fragment, normally read from a session key. The path is expected to be relative to the session directory but it works with an absolute URI, in this case it returns the same file than Gio.File.new_for_uri. It is useful to keep only relative file paths in a session file to be able to copy the whole project without breaking references.
