compute(factor)
Parameters:factor (float) – the progress factor, between 0 and 1
Returns:a pointer to the computed value, or None if the computation was not successfull
Return type:GObject.Value

Computes the value between the interval boundaries given the progress factor

Unlike Clutter.Interval.compute_value (), this function will return a const pointer to the computed value

You should use this function if you immediately pass the computed value to another function that makes a copy of it, like GObject.Object.set_property ()
