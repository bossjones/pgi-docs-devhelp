Return type:Cogl.Material

Retrieves the material used as a render target for the offscreen buffer created by effect

You should only use the returned Cogl.Material when painting. The returned material might change between different frames.

get_texture()
Returns:a Cogl.Handle or %COGL_INVALID_HANDLE. The returned texture is owned by Clutter and it should not be modified or freed
Return type:object

Retrieves the texture used as a render target for the offscreen buffer created by effect

You should only use the returned texture when painting. The texture may change after Clutter.Effect ::pre-paint is called so the effect implementation should update any references to the texture after chaining-up to the parent’s pre_paint implementation. This can be used instead of Clutter.OffscreenEffect.get_target () when the effect subclass wants to paint using its own material.
