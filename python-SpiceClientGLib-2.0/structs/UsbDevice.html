class SpiceClientGLib.UsbDevice
get_description(format)
Parameters:format (str or None) – an optional printf() format string with positional parameters
Returns:a newly-allocated string holding the description, or None if failed
Return type:str

Get a string describing the device which is suitable as a description of the device for the end user. The returned string should be freed with GLib.free () when no longer needed.

The format positional parameters are the following: - ‘%%1$s’ manufacturer - ‘%%2$s’ product - ‘%%3$s’ descriptor (a [vendor_id:product_id] string) - ‘%%4$d’ bus - ‘%%5$d’ address

(the default format string is “%%s %%s %%s at %%d-%%d”)
