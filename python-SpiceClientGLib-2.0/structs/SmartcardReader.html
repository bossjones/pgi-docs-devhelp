class SpiceClientGLib.SmartcardReader
insert_card()
Returns:True if insertion of a card was successfully simulated, False otherwise
Return type:bool

Simulates insertion of a smartcard in the software smartcard reader reader. If reader is not a software smartcard reader, False will be returned.

is_software()
Returns:True if reader is a software (emulated) smartcard reader, False otherwise
Return type:bool

Tests if reader is a software (emulated) smartcard reader.

remove_card()
Returns:True if removal of a card was successfully simulated, False otherwise
Return type:bool

Simulates removal of a smartcard from the software smartcard reader reader. If reader is not a software smartcard reader, False will be returned.
