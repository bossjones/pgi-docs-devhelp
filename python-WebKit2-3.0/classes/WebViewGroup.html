static new(name)
Parameters:name (str or None) – the name of the group
Returns:a new WebKit2.WebViewGroup
Return type:WebKit2.WebViewGroup

Creates a new WebKit2.WebViewGroup with the given name. If name is None a unique identifier name will be created automatically. The newly created WebKit2.WebViewGroup doesn’t contain any WebKit2.WebView, web views are added to the new group when created with WebKit2.WebView.new_with_group () passing the group.

add_user_style_sheet(source, base_uri, whitelist, blacklist, injected_frames)
Parameters:
  • source (str) – the source of the style_sheet to inject
  • base_uri (str or None) – the base URI to use when processing the style_sheet contents or None for about :blank
  • whitelist ([str] or None) – a whitelist of URI patterns or None
  • blacklist ([str] or None) – a blacklist of URI patterns or None
  • injected_frames (WebKit2.InjectedContentFrames) – a WebKit2.InjectedContentFrames describing to which frames the style_sheet should apply

Inject an external style sheet into pages. It is possible to only apply the style sheet to some URIs by passing non-null values for whitelist or blacklist. Passing a None whitelist implies that all URIs are on the whitelist. The style sheet is applied if a URI matches the whitelist and not the blacklist. URI patterns must be of the form [protocol]://[host]/[path] where the host and path components can contain the wildcard character (‘*’) to represent zero or more other characters.

set_settings(settings)
Parameters:settings (WebKit2.Settings) – a WebKit2.Settings

Sets a new WebKit2.Settings for the WebKit2.WebViewGroup. The settings will affect to all the WebKit2.WebView s of the group. WebKit2.WebViewGroup s always have a WebKit2.Settings so if you just want to modify a setting you can use WebKit2.WebViewGroup.get_settings () and modify the returned WebKit2.Settings instead. Setting the same WebKit2.Settings multiple times doesn’t have any effect. You can monitor the settings of a WebKit2.WebViewGroup by connecting to the notify ::settings signal of group.
