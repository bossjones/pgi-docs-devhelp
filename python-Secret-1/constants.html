Constants

  • Secret.COLLECTION_DEFAULT

Details

Secret.COLLECTION_DEFAULT = 'default'

An alias to the default collection. This can be passed to Secret.password_store () Secret.Collection.for_alias ().

Secret.COLLECTION_SESSION = 'session'

An alias to the session collection, which will be cleared when the user ends the session. This can be passed to Secret.password_store (), Secret.Collection.for_alias () or similar functions.
