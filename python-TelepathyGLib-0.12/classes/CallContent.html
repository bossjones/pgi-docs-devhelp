send_tones_async(tones, cancellable, callback, *user_data)
Parameters:
  • tones (str) – a string representation of one or more DTMF events.
  • cancellable (Gio.Cancellable or None) – optional Gio.Cancellable object, None to ignore
  • callback (Gio.AsyncReadyCallback or None) – a callback to call when the operation finishes
  • user_data (object) – data to pass to callback

Send tones DTMF code on self content. self must have the TelepathyGLib.IFACE_CALL_CONTENT_INTERFACE_DTMF interface.

If DTMF tones are already being played, this request is queued.

New in version 0.17.5.
