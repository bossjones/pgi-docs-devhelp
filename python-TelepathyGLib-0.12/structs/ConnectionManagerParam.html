class TelepathyGLib.ConnectionManagerParam

Structure representing a connection manager parameter.

Since 0.19.1, accessing the fields of this struct is deprecated, and they are no longer documented here. Use the accessors TelepathyGLib.ConnectionManagerParam.get_name (), TelepathyGLib.ConnectionManagerParam.get_dbus_signature (), TelepathyGLib.ConnectionManagerParam.is_required (), TelepathyGLib.ConnectionManagerParam.is_required_for_registration (), TelepathyGLib.ConnectionManagerParam.is_secret (), TelepathyGLib.ConnectionManagerParam.is_dbus_property (), TelepathyGLib.ConnectionManagerParam.get_default (), TelepathyGLib.ConnectionManagerParam.dup_default_variant () instead.
