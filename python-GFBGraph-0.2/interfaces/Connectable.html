default_parse_connected_data(payload)
Parameters:payload (str) – a const str with the response string from the Facebook Graph API.
Raises:GLib.GError
Returns:a newly-allocated GLib.List of GFBGraph.Node with the same GObject.Type as self.
Return type:[GFBGraph.Node]

In most cases, GFBGraph.Connectable implementers can use this function in order to parse the response when a GFBGraph.Node.get_connection_nodes () is executed and the GFBGraph.Connectable.parse_connected_data () was called.

Normally, Facebook Graph API returns the connections in the same way, using JSON objects, with a root object called “data”.
