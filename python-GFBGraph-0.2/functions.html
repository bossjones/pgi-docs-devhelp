GFBGraph.new_rest_call(authorizer)
Parameters:authorizer (GFBGraph.Authorizer) – a GFBGraph.Authorizer.
Returns:a new Rest.ProxyCall or None in case of error.
Return type:Rest.ProxyCall

Create a new Rest.ProxyCall pointing to the Facebook Graph API url (https://graph.facebook.com) and processed by the authorizer to allow queries.
