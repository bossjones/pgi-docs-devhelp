Class Details

class Atk.ObjectFactory(**kwargs)

Bases: GObject.Object

This class is the base object class for a factory used to create an accessible object for a specific GObject.Type. The function Atk.Registry.set_factory_type () is normally called to store in the registry the factory type to be used to create an accessible of a particular GObject.Type.

invalidate()

Inform factory that it is no longer being used to create accessibles. When called, factory may need to inform Atk.Objects which it has created that they need to be re-instantiated. Note: primarily used for runtime replacement of Atk.ObjectFactorys in object registries.
