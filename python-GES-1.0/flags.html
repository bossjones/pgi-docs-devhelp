class GES.PipelineFlags

Bases: GObject.GFlags

The various modes the GES.Pipeline can be configured to.

AUDIO_PREVIEW = 1

output audio to the soundcard

VIDEO_PREVIEW = 2

output video to the screen

FULL_PREVIEW = 3

output audio/video to soundcard/screen (default)

RENDER = 4

render timeline (forces decoding)

SMART_RENDER = 8

render timeline (tries to avoid decoding/reencoding)

class GES.TrackType

Bases: GObject.GFlags

Types of content handled by a track. If the content is not one of GES.TrackType.AUDIO, GES.TrackType.VIDEO or GES.TrackType.TEXT, the user of the GES.Track must set the type to GES.TrackType.CUSTOM.

GES.TrackType.UNKNOWN is for internal purposes and should not be used by users

UNKNOWN = 1

A track of unknown type (i.e. invalid)

AUDIO = 2

An audio track

VIDEO = 4

A video track

TEXT = 8

A text (subtitle) track

CUSTOM = 16

A custom-content track
