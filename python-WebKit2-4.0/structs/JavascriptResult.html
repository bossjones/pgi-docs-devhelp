class WebKit2.JavascriptResult
get_global_context()
Returns:the JavaScriptCore.GlobalContext for the WebKit2.JavascriptResult
Return type:JavaScriptCore.GlobalContext

Get the global Javascript context that should be used with the JavaScriptCore.Value returned by WebKit2.JavascriptResult.get_value ().

get_value()
Returns:the JavaScriptCore.Value of the WebKit2.JavascriptResult
Return type:JavaScriptCore.Value

Get the value of js_result. You should use the JavaScriptCore.GlobalContext returned by WebKit2.JavascriptResult.get_global_context () to use the JavaScriptCore.Value.

ref()
Returns:The passed in WebKit2.JavascriptResult
Return type:WebKit2.JavascriptResult

Atomically increments the reference count of js_result by one. This function is MT-safe and may be called from any thread.

unref()

Atomically decrements the reference count of js_result by one. If the reference count drops to 0, all memory allocated by the WebKit2.JavascriptResult is released. This function is MT-safe and may be called from any thread.
