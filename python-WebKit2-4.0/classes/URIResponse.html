get_content_length()
Returns:the expected content length of response.
Return type:int

Get the expected content length of the WebKit2.URIResponse. It can be 0 if the server provided an incorrect or missing Content-Length.

get_http_headers()
Returns:a Soup.MessageHeaders with the HTTP headers of response or None if response is not an HTTP response.
Return type:Soup.MessageHeaders

Get the HTTP headers of a WebKit2.URIResponse as a Soup.MessageHeaders.

New in version 2.6.

get_mime_type()
Returns:the MIME type of the WebKit2.URIResponse
Return type:str
get_status_code()
Returns:the status code of response
Return type:int

Get the status code of the WebKit2.URIResponse as returned by the server. It will normally be a Soup.KnownStatusCode, for example Soup.Status.OK, though the server can respond with any unsigned integer.

get_suggested_filename()
Returns:the suggested filename or None if the ‘Content-Disposition’ HTTP header is not present.
Return type:str

Get the suggested filename for response, as specified by the ‘Content-Disposition’ HTTP header, or None if it’s not present.
