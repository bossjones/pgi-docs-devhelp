set_persistent_storage(filename, storage)
Parameters:
  • filename (str) – the filename to read to/write from
  • storage (WebKit2.CookiePersistentStorage) – a WebKit2.CookiePersistentStorage

Set the filename where non-session cookies are stored persistently using storage as the format to read/write the cookies. Cookies are initially read from filename to create an initial set of cookies. Then, non-session cookies will be written to filename when the WebKit2.CookieManager ::changed signal is emitted. By default, cookie_manager doesn’t store the cookies persistenly, so you need to call this method to keep cookies saved across sessions.
