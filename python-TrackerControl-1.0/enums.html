class TrackerControl.MinerManagerError

Bases: GObject.GEnum

Enumeration values used in errors returned by the TrackerControl.MinerManager API.

New in version 0.8.

NOT_AVAILABLE = 0

The miner in question is not active and can so can not be used.

NOENT = 1

The resource that the miner is handling (for example a file or URI) does not exist.
