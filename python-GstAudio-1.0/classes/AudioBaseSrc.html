Class Details

class GstAudio.AudioBaseSrc(**kwargs)

Bases: GstBase.PushSrc

This is the base class for audio sources. Subclasses need to implement the ::create-ringbuffer vmethod. This base class will then take care of reading samples from the ringbuffer, synchronisation and flushing.

create_ringbuffer()
Returns:The new ringbuffer of src.
Return type:GstAudio.AudioRingBuffer

Create and return the GstAudio.AudioRingBuffer for src. This function will call the ::create-ringbuffer vmethod and will set src as the parent of the returned buffer (see Gst.Object.set_parent ()).

get_provide_clock()
Returns:True if src will provide a clock.
Return type:bool

Queries whether src will provide a clock or not. See also GstAudio.AudioBaseSrc.set_provide_clock.

get_slave_method()
Returns:The current slave method used by src.
Return type:GstAudio.AudioBaseSrcSlaveMethod

Get the current slave method used by src.

set_provide_clock(provide)
Parameters:provide (bool) – new state

Controls whether src will provide a clock or not. If provide is True, Gst.Element.provide_clock () will return a clock that reflects the datarate of src. If provide is False, Gst.Element.provide_clock () will return None.

set_slave_method(method)
Parameters:method (GstAudio.AudioBaseSrcSlaveMethod) – the new slave method

Controls how clock slaving will be performed in src.

do_create_ringbuffer()
Type:virtual
Returns:The new ringbuffer of src.
Return type:GstAudio.AudioRingBuffer

Create and return the GstAudio.AudioRingBuffer for src. This function will call the ::create-ringbuffer vmethod and will set src as the parent of the returned buffer (see Gst.Object.set_parent ()).
