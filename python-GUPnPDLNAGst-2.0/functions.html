utils_information_from_discoverer_info(info)

Details

GUPnPDLNAGst.utils_information_from_discoverer_info(info)
Parameters:info (GstPbutils.DiscovererInfo) – A GstPbutils.DiscovererInfo object.
Returns:A GUPnPDLNA.Information. Unref it when done with it.
Return type:GUPnPDLNA.Information

This utility function creates a GUPnPDLNA.Information from GstPbutils.DiscovererInfo. Its purpose is mainly to pass its result to GUPnPDLNA.ProfileGuesser.guess_profile_from_info () to avoid rediscovering of a URI. Created GUPnPDLNA.Information takes a reference on info.
