class Gst.AtomicQueue

The Gst.AtomicQueue object implements a queue that can be used from multiple threads without performing any blocking operations.

static new(initial_size)
Parameters:initial_size (int) – initial queue size
Returns:a new Gst.AtomicQueue
Return type:Gst.AtomicQueue

Create a new atomic queue instance. initial_size will be rounded up to the nearest power of 2 and used as the initial size of the queue.

length()
Returns:the number of elements in the queue.
Return type:int

Get the amount of items in the queue.

peek()
Returns:the head element of queue or None when the queue is empty.
Return type:object or None

Peek the head element of the queue without removing it from the queue.

pop()
Returns:the head element of queue or None when the queue is empty.
Return type:object

Get the head element of the queue.

push(data)
Parameters:data (object) – the data

Append data to the tail of the queue.

ref()

Increase the refcount of queue.

unref()

Unref queue and free the memory when the refcount reaches 0.
