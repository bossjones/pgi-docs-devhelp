class Gst.BufferList

Buffer lists are an object containing a list of buffers.

Buffer lists are created with Gst.BufferList.new () and filled with data using a Gst.BufferList.insert ().

Buffer lists can be pushed on a srcpad with Gst.Pad.push_list (). This is interesting when multiple buffers need to be pushed in one go because it can reduce the amount of overhead for pushing each buffer individually.

static new()
Returns:the new Gst.BufferList. gst_buffer_list_unref() after usage.
Return type:Gst.BufferList

Creates a new, empty Gst.BufferList. The caller is responsible for unreffing the returned Gst.BufferList.

Free-function: gst_buffer_list_unref

static new_sized(size)
Parameters:size (int) – an initial reserved size
Returns:the new Gst.BufferList. gst_buffer_list_unref() after usage.
Return type:Gst.BufferList

Creates a new, empty Gst.BufferList. The caller is responsible for unreffing the returned Gst.BufferList. The list will have size space preallocated so that memory reallocations can be avoided.

Free-function: gst_buffer_list_unref

foreach(func, *user_data)
Parameters:
  • func (Gst.BufferListFunc) – a Gst.BufferListFunc to call
  • user_data (object) – user data passed to func
Returns:

True when func returned True for each buffer in list or when list is empty.
Return type:

bool

Call func with data for each buffer in list.

func can modify the passed buffer pointer or its contents. The return value of func define if this function returns or if the remaining buffers in the list should be skipped.

get(idx)
Parameters:idx (int) – the index
Returns:the buffer at idx in group or None when there is no buffer. The buffer remains valid as long as list is valid.
Return type:Gst.Buffer or None

Get the buffer at idx.

insert(idx, buffer)
Parameters:
  • idx (int) – the index
  • buffer (Gst.Buffer) – a Gst.Buffer

Insert buffer at idx in list. Other buffers are moved to make room for this new buffer.

A -1 value for idx will append the buffer at the end.

length()
Returns:the number of buffers in the buffer list
Return type:int

Returns the number of buffers in list.

remove(idx, length)
Parameters:
  • idx (int) – the index
  • length (int) – the amount to remove

Remove length buffers starting from idx in list. The following buffers are moved to close the gap.
