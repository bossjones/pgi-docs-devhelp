insert(position)
Parameters:position (int) – position in the playlist
Returns:an unset TotemPlParser.PlaylistIter for returning the location
Return type:iter: TotemPlParser.PlaylistIter

Inserts a new empty element to playlist at position, and modifies iter so it points to it. To fill in values, you need to call TotemPlParser.Playlist.set () or TotemPlParser.Playlist.set_value ().

position may be minor than 0 to prepend elements, or bigger than the current playlist size to append elements.
