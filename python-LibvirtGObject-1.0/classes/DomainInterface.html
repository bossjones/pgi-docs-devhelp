get_stats()
Raises:GLib.GError
Returns:the stats or None in case of error. The returned object should be unreffed with GObject.Object.unref () when no longer needed.
Return type:LibvirtGObject.DomainInterfaceStats

This function returns network interface stats. Individual fields within the stats structure may be returned as -1, which indicates that the hypervisor does not support that particular statistic.
