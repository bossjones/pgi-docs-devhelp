Details

class GLib.Node

The GLib.Node struct represents one node in a [n-ary tree][glib-N-ary-Trees].

child_index(data)
Parameters:data (object) – the data to find
Returns:the index of the child of node which contains data, or -1 if the data is not found
Return type:int

Gets the position of the first child of a GLib.Node which contains the given data.

child_position(child)
Parameters:child (GLib.Node) – a child of node
Returns:the position of child with respect to its siblings
Return type:int

Gets the position of a GLib.Node with respect to its siblings. child must be a child of node. The first child is numbered 0, the second 1, and so on.

depth()
Returns:the depth of the GLib.Node
Return type:int

Gets the depth of a GLib.Node.

If node is None the depth is 0. The root node has a depth of 1. For the children of the root node the depth is 2. And so on.

destroy()

Removes root and its children from the tree, freeing any memory allocated.

is_ancestor(descendant)
Parameters:descendant (GLib.Node) – a GLib.Node
Returns:True if node is an ancestor of descendant
Return type:bool

Returns True if node is an ancestor of descendant. This is true if node is the parent of descendant, or if node is the grandparent of descendant etc.

max_height()
Returns:the maximum height of the tree beneath root
Return type:int

Gets the maximum height of all branches beneath a GLib.Node. This is the maximum distance from the GLib.Node to all leaf nodes.

If root is None, 0 is returned. If root has no children, 1 is returned. If root has children, 2 is returned. And so on.

n_children()
Returns:the number of children of node
Return type:int

Gets the number of children of a GLib.Node.

n_nodes(flags)
Parameters:flags (GLib.TraverseFlags) – which types of children are to be counted, one of GLib.TraverseFlags.ALL, GLib.TraverseFlags.LEAVES and GLib.TraverseFlags.NON_LEAVES
Returns:the number of nodes in the tree
Return type:int

Gets the number of nodes in a tree.

reverse_children()

Reverses the order of the children of a GLib.Node. (It doesn’t change the order of the grandchildren.)

unlink()

Unlinks a GLib.Node from a tree, resulting in two separate trees.
