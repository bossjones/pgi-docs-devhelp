Class Details

class WebKit.WebNavigationAction(**kwargs)

Bases: GObject.Object

WebKit.WebNavigationAction is used in signals to provide details about what led the navigation to happen. This includes, for instance, if the user clicked a link to start that navigation, and what mouse button was used.

get_button()
Returns:the mouse button used to click
Return type:int

The GTK+ identifier for the mouse button used to click. Notice that GTK+ button values are 1, 2 and 3 for left, middle and right buttons, so they are DOM button values +1. If the action was not initiated by a mouse click the value will be -1.

New in version 1.0.3.
