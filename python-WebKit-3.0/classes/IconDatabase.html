Class Details

class WebKit.IconDatabase(**kwargs)

Bases: GObject.Object

WebKit.IconDatabase provides access to website icons, as shown in tab labels, window captions or bookmarks. All views share the same icon database.

The icon database is enabled by default and stored in ~/.cache/webkit/icondatabase, depending on XDG_CACHE_HOME.

WebKit will automatically look for available icons in link elements on opened pages as well as an existing favicon.ico and load the images found into the memory cache if possible. The signal “icon-loaded” will be emitted when any icon is found and loaded. Old Icons are automatically cleaned up after 4 days.

WebKit.IconDatabase.set_path () can be used to change the location of the database and also to disable it by passing None.

If WebKit.WebSettings ::enable-private-browsing is True new icons won’t be added to the database on disk and no existing icons will be deleted from it.

clear()

Clears all icons from the database.

New in version 1.3.13.

Deprecated since version 1.8: Use WebKit.FaviconDatabase.clear () instead.

get_icon_pixbuf(page_uri)
Parameters:page_uri (str) – URI of the page containing the icon
Returns:a new reference to a GdkPixbuf.Pixbuf, or None
Return type:GdkPixbuf.Pixbuf

Obtains a GdkPixbuf.Pixbuf of the favicon for the given page URI, or a default icon if there is no icon for the given page. Use WebKit.IconDatabase.get_icon_uri () if you need to distinguish these cases. Usually you want to connect to WebKit.IconDatabase ::icon-loaded and call this method in the callback.

The pixbuf will have the largest size provided by the server and should be resized before it is displayed. See also WebKit.WebView.get_icon_pixbuf ().

New in version 1.3.13.

Deprecated since version 1.8: Use WebKit.FaviconDatabase.try_get_favicon_pixbuf () instead.

get_icon_uri(page_uri)
Parameters:page_uri (str) – URI of the page containing the icon
Returns:a newly allocated URI for the favicon, or None
Return type:str

Obtains the URI for the favicon for the given page URI. See also WebKit.WebView.get_icon_uri ().

New in version 1.3.13.

Deprecated since version 1.8: Use WebKit.FaviconDatabase.get_favicon_uri () instead.

set_path(path)
Parameters:path (str) – an absolute path to the icon database folder

Specifies the absolute path to the database folder on disk.

Passing None or “” disables the icon database.

New in version 1.3.13.

Signal Details

WebKit.IconDatabase.signals.icon_loaded(icon_database, frame, frame_uri)
Signal Name:

icon-loaded
Flags:

RUN_LAST
Parameters:
  • icon_database (WebKit.IconDatabase) – The object which received the signal
  • frame (WebKit.WebFrame) – the frame containing the icon
  • frame_uri (str) – the URI of the frame containing the icon

This signal is emitted when a favicon is available for a page, or a child frame. See WebKit.WebView ::icon-loaded if you only need the favicon for the main frame of a particular WebKit.WebView.

New in version 1.3.13.

Deprecated since version 1.8: Use WebKit.FaviconDatabase ::icon-loaded instead.
