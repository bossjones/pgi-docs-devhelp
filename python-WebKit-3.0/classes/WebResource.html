Class Details

class WebKit.WebResource(**kwargs)

Bases: GObject.Object

A web resource encapsulates the data of the download as well as the URI, MIME type and frame name of the resource.

static new(data, size, uri, mime_type, encoding, frame_name)
Parameters:
  • data (str) – the data to initialize the WebKit.WebResource
  • size (int) – the length of data
  • uri (str) – the URI of the WebKit.WebResource
  • mime_type (str) – the MIME type of the WebKit.WebResource
  • encoding (str) – the text encoding name of the WebKit.WebResource
  • frame_name (str) – the frame name of the WebKit.WebResource
Returns:

a new WebKit.WebResource
Return type:

WebKit.WebResource

Returns a new WebKit.WebResource. The encoding can be None. The frame_name argument can be used if the resource represents contents of an entire HTML frame, otherwise pass None.

New in version 1.1.14.

Signal Details

WebKit.WebResource.signals.content_length_received(web_resource, length_received)
Signal Name:

content-length-received
Flags:

RUN_LAST
Parameters:
  • web_resource (WebKit.WebResource) – The object which received the signal
  • length_received (int) – the amount of data received since the last signal emission

Emitted when new resource data has been received. The length_received variable stores the amount of bytes received since the last time this signal was emitted. This is useful to provide progress information about the resource load operation.

New in version 1.7.5.
