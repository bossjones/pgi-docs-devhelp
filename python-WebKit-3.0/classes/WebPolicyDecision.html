class WebKit.WebPolicyDecision(**kwargs)

Bases: GObject.Object

WebKit.WebPolicyDecision objects are given to the application on signal emissions that deal with policy decisions, such as if a new window should be opened, or if a given navigation should be allowed. The application uses it to tell the engine what to do.

download()

Will send the DOWNLOAD decision to the policy implementer.

New in version 1.0.3.

ignore()

Will send the IGNORE decision to the policy implementer.

New in version 1.0.3.

use()

Will send the USE decision to the policy implementer.

New in version 1.0.3.
