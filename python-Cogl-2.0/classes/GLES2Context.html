class Cogl.GLES2Context

Bases: Cogl.Object

Represents an OpenGLES 2.0 api context used as a sandbox for OpenGLES 2.0 state. This is comparable to an EGLContext for those who have used OpenGLES 2.0 with EGL before.

New in version 1.12.
