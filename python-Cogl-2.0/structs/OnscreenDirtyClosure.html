class Cogl.OnscreenDirtyClosure

An opaque type that tracks a Cogl.OnscreenDirtyCallback and associated user data. A Cogl.OnscreenDirtyClosure pointer will be returned from Cogl.Onscreen.add_dirty_callback () and it allows you to remove a callback later using Cogl.Onscreen.remove_dirty_callback ().

New in version 1.16.
