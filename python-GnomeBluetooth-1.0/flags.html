class GnomeBluetooth.Type

Bases: GObject.GFlags

The type of a Bluetooth device. See also GnomeBluetooth.TYPE_INPUT and GnomeBluetooth.TYPE_AUDIO

ANY = 1

any device, or a device of an unknown type

PHONE = 2

a telephone (usually a cell/mobile phone)

MODEM = 4

a modem

COMPUTER = 8

a computer, can be a laptop, a wearable computer, etc.

NETWORK = 16

a network device, such as a router

HEADSET = 32

a headset (usually a hands-free device)

HEADPHONES = 64

headphones (covers two ears)

OTHER_AUDIO = 128

another type of audio device

KEYBOARD = 256

a keyboard

MOUSE = 512

a mouse

CAMERA = 1024

a camera (still or moving)

PRINTER = 2048

a printer

JOYPAD = 4096

a joypad, joystick, or other game controller

TABLET = 8192

a drawing tablet

VIDEO = 16384

a video device, such as a webcam

REMOTE_CONTROL = 32768

a remote control

SCANNER = 65536

a scanner

DISPLAY = 131072

a display

WEARABLE = 262144

a wearable computer

TOY = 524288

a toy or game
